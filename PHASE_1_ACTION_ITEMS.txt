TROUBLESHOOTING:
═════════════════════════════════════════════════════════════════════════════════

Q: Getting "Failed opening required '_bootstrap.php'" error?
A: File is not in the right location. Copy it to your project root.

Q: Getting database connection errors?
A: Check config/config.php has correct DB_HOST, DB_USER, DB_PASS, DB_NAME

Q: Getting "include path" errors?
A: Make sure config/config.php exists and has all required constants defined.

Q: Getting session errors?
A: Sessions should be auto-started by bootstrap. Check /tmp permissions for session files.

Q: Want to verify file was copied correctly?
A: Run: head -10 /path/to/your/project/_bootstrap.php
   Should show PHP comments and function definitions.
