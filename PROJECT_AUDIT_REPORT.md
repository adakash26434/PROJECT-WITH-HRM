## ⚠️ ISSUES FOUND & FIXES

### 🔴 CRITICAL ISSUES

#### 1. **Missing Bootstrap File Error**
**Status:** CRITICAL (Production-breaking)

```
Error: Failed opening required '/home/bandanas/_bootstrap.php'
Location: index.php line 6
```

**Root Cause:** The `_bootstrap.php` file is missing or not properly referenced. The project uses a bootstrap pattern but the file doesn't exist.

**Fix:** Create the bootstrap file:
```php
// _bootstrap.php — यो file create गर्नुहोस्
<?php
// Database connection
require_once __DIR__ . '/includes/database.local.php';  // Auto-generated by installer

// Authentication
require_once __DIR__ . '/includes/auth.php';

// Helpers
require_once __DIR__ . '/includes/helpers.php';

// Global config
define('SITE_URL', !empty($_SERVER['HTTPS']) ? 'https://' : 'http://' . $_SERVER['HTTP_HOST']);
define('UPLOAD_DIR', __DIR__ . '/assets/uploads/');
```

### 🟡 MEDIUM PRIORITY ISSUES

#### 3. **Inconsistent Font Sizing**
**Status:** MEDIUM (Mobile-first fix needed)

**Current State:**
- Base font: 0.875rem (14px) ✅ Good
- But some pages have inline `font-size` overrides
- Mobile scaling uses `clamp()` inconsistently

**Recommendation:**
Create a comprehensive `font-system.css`:
```css
html { font-size: 16px; } /* 1rem = 16px */
body { font-size: 0.875rem; } /* 14px base */

h1 { font-size: clamp(1.75rem, 5vw, 2.5rem); }
h2 { font-size: clamp(1.5rem, 4vw, 2rem); }
h3 { font-size: clamp(1.25rem, 3vw, 1.5rem); }
```

```
assets/css/
├── design-tokens.css          ← 🎨 Central color/font defs
├── theme-overrides-v4.css     ← 🔄 Admin overrides
├── universal.css              ← 📐 Base typography
├── style.css                  ← ⚠️ 340KB — BLOATED
├── coop-core.css              ← Public page styles
├── public-modern.css          ← Alternative public (?)
├── header-v2.css              ← Header specific
├── admin-tokens.css           ← Admin specific
├── components.css             ← UI components
└── [10+ more...]              ← Fragmented
```

**Problem:** Too many overlapping files. Hard to maintain.

**Solution:** Consolidate into:
```
assets/css/
├── design-tokens.css   ← Central variables
├── base.css            ← Typography, forms, buttons
├── layout.css          ← Grid, flexbox, containers
├── public.css          ← Public-specific
├── admin.css           ← Admin-specific
├── member.css          ← Member portal-specific
└── theme.css           ← Color overrides
```
